# Generic TypeScript App

This is a small example app showing how a *consumer* project integrates SharkCraft.

The real source code lives under `src/`. SharkCraft project knowledge (rules,
paths, templates, docs) lives under `sharkcraft/` and is queried by the CLI/MCP.

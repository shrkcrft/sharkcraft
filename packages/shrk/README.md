# shrk

Unscoped install path for the SharkCraft CLI.

```bash
npx shrk@alpha init
npx shrk@alpha doctor
npx shrk@alpha task "generate a service"
```

`shrk` is a thin wrapper around [`@shrkcrft/cli`](https://www.npmjs.com/package/@shrkcrft/cli) — same surface, same flags, same exit codes. Use whichever name you prefer; the wrapper exists so the bare `npx shrk` form resolves on the public registry.

See the [main repository](https://github.com/shrkcrft/sharkcraft) for full docs.
